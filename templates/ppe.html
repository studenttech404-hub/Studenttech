📜 Overview

This project focuses on building a Personal Protective Equipment (PPE) Detection System to ensure workplace safety by identifying PPE compliance from images or video feeds.

We utilized the RoboUniverse dataset and further annotated, augmented, and trained the data to create a highly accurate model for real-world PPE monitoring.

📂 Project Workflow

🔹 Dataset AcquisitionSourced base dataset from RoboUniverse.
🔹 AnnotationAnnotated images using Roboflow.
🔹 AugmentationApplied rotation, brightness, flipping, and scaling.
🔹 TrainingTrained on Google Colab using YOLOv8.
🔹 Early StoppingTraining stopped when mAP@0.5 dropped continuously for 5 epochs.
🔹 DeploymentOptimized model ready for edge device or server integration.
