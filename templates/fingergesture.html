✋ Finger Gesture Drawing 🎨

Draw on your screen using your hand gestures! Point, switch colors, clear the canvas – all with your fingers. 🖐️ No mouse needed. Just ✋ and vibes.

🌟 Features

  • 🎨 Draw in real-time using your index finger
  • 🔴 Change colors by showing 1, 2 or 3 fingers:
    • 1 finger → Red
    • 2 fingers → Green
    • 3 fingers → Blue
  • 🖐️ Show all 5 fingers to clear the screen
  • ✊ Make a fist to pause drawing
  • 🖥️ Built with OpenCV + Mediapipe

🎮 Controls

  • 1 Finger → 🔴 Red
  • 2 Fingers → 🟢 Green
  • 3 Fingers → 🔵 Blue
  • 5 Fingers → Clear screen
  • Fist → Pause drawing
  • q → Quit the app

📝 Notes

This project has way too many if statements... But hey, if it works, don’t touch it. 🫡

💻 Tech Stack

  • 🐍 Python
  • 🖼️ OpenCV
  • 🖐️ Mediapipe

🚀 How to Run

git clone https://github.com/emaaanbutt/Finger-Gesture-Drawing.git
cd Finger-Gesture-Drawing
pip install -r requirements.txt
python app.py
