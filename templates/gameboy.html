🎮 Classic GameBoy Project

A retro-inspired gaming platform crafted in modern C++ with SFML and powered by solid OOP principles. Featuring three nostalgic classics: Snake Xenia, Wordle, and Hangman.

🎯 Features

  • 🕹️ Multiple Games – Snake, Wordle, and Hangman
  • 👤 Player Profiles – Manage accounts and track scores
  • 📱 GameBoy-like Console Menu – Navigate through games with retro UI
  • 🎨 SFML Graphics – 2D rendering and animations
  • 🎼 Audio Integration – Retro sound effects for immersion
  • 🎵 Game Over Screen – Dramatic sound effects
  • 🧠 Modular OOP Design – Clean, extensible, and maintainable

🛠️ Installation

git clone https://github.com/rauf17/GameBoy_Project.git
cd GameBoy_Project
# Find and run the executable file:
i230591_OOP_GameBoy_Project.exe

▶️ Navigate to the GameBoy_Project folder and run i230591_OOP_GameBoy_Project.exe to play.

🧰 Games Included

  • 🐍 Snake Xenia – Grow after eating mushrooms, avoid collisions, collect coins for bonus.
  • 🧩 Wordle Clone – Guess the 5-letter word with limited tries and feedback tiles.
  • 💀 Hangman – Guess the word before the full man is drawn, hints included.

🧱 Object-Oriented Concepts Used

  • 🔁 Inheritance – All games inherit from an abstract Game class with a unified Run() method.
  • 🔄 Polymorphism – GameBoy runs any game via a Game* pointer.
  • 🔗 Composition – GameBoy composed of MainMenu, Player, Leaderboard, Game.
  • 🧺 Aggregation – Multiple games share leaderboard & player access without ownership.
  • 🧊 Abstraction – Abstract Game defines clear interfaces for all games.

🕹️ How to Play

  • 🐍 Snake Xenia: Arrow keys/WASD to move, eat mushrooms, avoid walls, coins give bonus.
  • 🧩 Wordle: Guess 5-letter word, feedback via tiles, input via keyboard or mouse.
  • 💀 Hangman: Guess letters, wrong guesses draw the man, limited lives, hints available.

⚙️ Development

  • 💬 Language: C++
  • 🧰 Library: SFML (Graphics, Audio, Window, Events)
  • 🧱 Paradigm: Pure Object-Oriented Programming
  • 💻 Tools: Visual Studio, Git, SFML
