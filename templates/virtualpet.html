🐾 PusheenPal

PusheenPal is a wholesome little desktop companion built using Electron.js! Choose hats 🎩, eat snacks 🍣, paint art 🎨, take naps 😴, and even celebrate with confetti 🎊 — all with your adorable cat buddy, Pusheen 💖.

✨ Features

  • 🍕 Eat Time – Feed your chubby cat with some delicious treats!
  • 🎨 Paint Mode – Watch Pusheen get artsy on the canvas.
  • 🎩 Hat Collection – Try on quirky hats and find the perfect look.
  • 😴 Sleepy Time – Let Pusheen nap with soft snoring sounds in the background.
  • 🎊 Confetti Celebration – Tap the button and let the confetti rain!

⚙️ Setup & Installation

🧁 Quick install:

git clone https://github.com/emaaanbutt/PusheenPal---Virtual-Pet.git
cd pusheenpal
npm install
npm start

🧁 How to Run:

  1. Clone the repo
  2. Run npm install to get node_modules
  3. Start the app with: npm start
